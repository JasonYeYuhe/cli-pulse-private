### en-US (primary)

```
v1.21.0 — smoother launch, sharper iPad routing, widget always fresh.

- Faster cold start: telemetry init moved off the launch thread on iOS and Mac.
- iPad: notification taps now reliably switch to the right tab.
- iOS home-screen widget refreshes every hour via silent push, even when the app is closed. Shows a "last updated" timestamp and a Smart Stack relevance score.
- Mac auto-update verifies the SHA-256 of downloaded DMGs without freezing the menu-bar popover.
- Receipt validation hardened to accept any current Apple Root CA chain — fixes a subset of TestFlight + sandbox receipts that were silently failing.
- Push tokens reconciled on every cold launch so notifications keep working after iOS reinstalls or reboots.
- Remote Control privacy consent dialog now reads cleanly in Simplified Chinese.
- Users with Traditional Chinese as their preferred system language now see Simplified Chinese throughout the app instead of falling back to English.
```

```
## v1.21.0 — long-tail audit fixes

This release rolls up the v1.21 multi-platform audit fixes. Same Mac
binary you'd download from the Mac App Store, distributed via
notarized DMG for the DEVID channel.

### What changed for you

- **Faster launch**: Sentry telemetry initialisation no longer runs on
  the launch thread, so cold-start latency stays bounded even on
  weak-network days.
- **Auto-update is snappier**: the SHA-256 fingerprint check on a
  downloaded DMG runs off the main thread. The menubar popover no
  longer hangs for 1-3 seconds while a 50 MB download is verified.
- **Receipt validation hardened**: the prior shipped Apple Root CA G3
  certificate body was malformed in source. Replaced with the verified
  G3 PEM and added Apple Root CA G2 + the original 2006 Apple Inc.
  Root so any receipt chain validates cleanly.

### Under the hood

- 19 long-tail items shipped (F-block backend, E-block Android,
  D-block Apple, M-block cross-cutting, G-block CI). Full breakdown
  in PROJECT_FIX_2026-05-15_v1.21_long_tail.md on the source repo.
- New CI gates: live migration replay (deferred), strict locale
  parity guard, version-drift gate.

### Notes for fresh installs

- LaunchAgent runtime is the same as v1.19.0 (still Swift-only,
  Phase 4D path).
- Same `~/Library/Group Containers/group.yyh.CLI-Pulse` AppGroup; if
  you're switching between MAS and DEVID channels, your local data
  carries over.

Distributed as a notarised .dmg signed by Developer ID Application:
Yuhe Ye (KHMK6Q3L3K).
```
