About

CLI Pulse is a developer app for monitoring usage, quotas, resets, sessions, and alerts across supported AI coding providers.

Supports Claude, Codex, Gemini, Cursor, Copilot, JetBrains AI, Ollama, Warp, Augment, and more. Available on macOS, iOS, watchOS, and Android (beta).
